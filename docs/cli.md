# CLI commands

## Vue CLI

### Create a project:

```
vue create hello-world
```

---

### Install dependencies

```
npm install
```

---

### Run local server

```
npm run serve
```

---

### Build for production

```
npm run build
```

## VuePress

### Run Server

```
yarn docs:dev
```

[localhost:8080](http://localhost:8080/)

---

### Build for production

```
yarn docs:build
```
